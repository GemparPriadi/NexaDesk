# NexaDesk - Modern Ticketing Helpdesk System

NexaDesk is a modern web-based ticketing and helpdesk management system built using Laravel 12, Livewire Volt, TailwindCSS, and Chart.js.

This project was developed to help users create and manage support tickets efficiently with a clean modern interface, realtime notifications, activity logs, and admin management features.

---

# ✨ Features

* Authentication Login & Register
* User Dashboard
* Admin Dashboard
* Ticket Management System
* Ticket Status Management
* Priority Ticket System
* Realtime Notifications
* Activity Logs
* Export Ticket PDF
* Dark Mode Support
* Responsive Modern UI
* SweetAlert Notification
* Chart Analytics
* User Profile Management

# 📦 Installation Guide

## 1. Clone Repository

```bash
git clone https://github.com/GemparPriadi/NexaDesk.git
```

## 2. Open Project Folder

```bash
cd NexaDesk
```

## 3. Install Composer Dependency

```bash
composer install
```

## 4. Install NPM Dependency

```bash
npm install
```

## 5. Setup Environment File

```bash
cp .env.example .env
```

## 6. Generate Laravel Key

```bash
php artisan key:generate
```

## 7. Setup Database

Create a database first, then configure your `.env`:

```env
DB_DATABASE=nexadesk
DB_USERNAME=root
DB_PASSWORD=
```

## 8. Run Migration

```bash
php artisan migrate
```

## 9. Storage Link

```bash
php artisan storage:link
```

## 10. Run Development Server

```bash
php artisan serve
```

## 11. Run Vite

```bash
npm run dev
```
